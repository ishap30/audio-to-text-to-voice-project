Audio to Text to Voice AI

A simple AI-powered web application that converts speech to text and text back to speech using OpenAI Whisper and Google Text-to-Speech (gTTS).
The project is built using Python, Flask, and modern web UI.

🚀 Features

🎤 Speech to Text

Upload or record audio

Transcription powered by OpenAI Whisper

🔊 Text to Speech

Convert typed text into natural speech

Generated using Google Text-to-Speech (gTTS)

🌐 Web Interface

Clean frontend using HTML, CSS, and JavaScript

Instant transcription and audio playback

🛠️ Tech Stack

Python

Flask

OpenAI Whisper

gTTS (Google Text-to-Speech)

HTML / CSS / JavaScript
